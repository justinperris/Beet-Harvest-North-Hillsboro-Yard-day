GitHub Auto-Sync Token

Paste your Personal Access Token here. This enables automatic updates to GitHub when you click Save Now on the main pages.



Instructions

Use a classic token with public_repo scope (easiest for public repos).

  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Generate new token
  3. Name it (e.g. "Piler Tools")
  4. Check public_repo
  5. Generate and copy the token
  6. Paste above and click Save Token

Once saved, "Save Now" on the info page or sign-in sheet will try to push updates to GitHub automatically.
